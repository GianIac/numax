## Invia il tuo progetto

Hai costruito qualcosa con Numax? Ci piacerebbe vederlo.

Apri una pull request che aggiunge una voce a questa pagina. Un side project di una riga,
un esperimento del weekend, un modulo demo, uno strumento, un esercizio per un corso —
qualsiasi cosa è benvenuta. L'unico requisito è che utilizzi Numax in qualche modo.

### Cosa includere

Aggiungi una nuova sezione con questo formato:

```markdown
### Nome del tuo progetto

**Repo:** [github.com/tu/tuo-progetto](https://github.com/tu/tuo-progetto)
**Cosa fa:** una o due frasi che descrivono cosa fa il progetto e quali funzionalità di Numax usa.
```

Tutto qui. Nessuno screenshot richiesto, nessun minimo di righe di codice, nessun deploy in produzione necessario.

### Come aprire la pull request

1. Forka il [repository numax](https://github.com/GianIac/numax).
2. Modifica `docs/nx-site/src/content/docs/it/showcase.md`.
3. Aggiungi la tua voce nella sezione appropriata qui sotto, o aggiungine una nuova se nessuna si adatta.
4. Apri una pull request con il titolo `showcase: aggiungi <nome del tuo progetto>`.

La mergeremo con grande onore !!
