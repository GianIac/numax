## GitHub Discussions

For now, the official place to ask questions, propose ideas, discuss roadmap
items, and share experiments is:

- [Numax GitHub Discussions](https://github.com/GianIac/numax/discussions)

More community spaces will arrive over time as the project grows. The goal is
to keep the first channels simple, useful, and close to the code.
