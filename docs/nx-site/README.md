# Numax Site

Documentation site scaffold for Numax, built with Astro Starlight.

## Local development

```bash
cd docs/nx-site
npm install
npm run dev
```

## GitHub comments

Blog posts can render GitHub Discussion comments through
[Giscus](https://giscus.app/).

The site is preconfigured for the `GianIac/numax` repository and the `Ideas`
Discussion category. To override those values locally, copy `.env.example` to
`.env` and edit it:

```bash
cp .env.example .env
```
